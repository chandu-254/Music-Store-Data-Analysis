# Music-Store-Data-Analysis
## Project Overview

The Music Store Data Analysis project focuses on analyzing a digital music store database using SQL to extract meaningful business insights. The dataset contains information about customers, invoices, tracks, albums, artists, genres, and sales.
The goal of this project is to improve understanding of customer behavior, sales trends, and revenue performance using structured query language.

## Objectives

* Analyze sales performance across different countries and cities
* Identify top-selling genres, artists, and tracks
* Understand customer purchasing behavior
* Calculate revenue trends and customer spending patterns
* Provide actionable insights to support business decisions
